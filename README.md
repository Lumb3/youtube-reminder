# üìå youtube-reminder

**youtube-reminder** is a lightweight and user-friendly Chrome extension that helps you remember YouTube videos you planned to watch later. Never miss a video again from your **Watch Later** list!  

---

## ‚öôÔ∏è Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** JavaScript (Chrome extension APIs)  

---

## ‚ú® Features

- Add videos from **YouTube Watch Later** directly to the extension.  
- Set customizable notification frequency.  
- Persistent reminders until the video is marked as watched.  
- Simple, minimalistic interface for easy use.

## üöÄ How to Install & Use

### 1. **Clone the repository**  
```bash
git clone https://github.com/Lumb3/youtube-reminder
````

Or download the project folder directly to your device.

### 2. **Enable Developer Mode in Chrome**

1. Open Chrome and go to **Extensions** ‚Üí **Manage Extensions**.
2. Turn on **Developer Mode**.
3. Click **Load unpacked** and select the downloaded project folder.

Voila! The extension is now installed.
